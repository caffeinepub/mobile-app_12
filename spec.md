# Specification

## Summary
**Goal:** Build a mobile-optimized e-commerce reseller application with product catalog, shopping cart, checkout flow, order management, and inventory tracking.

**Planned changes:**
- Create product catalog page with responsive grid displaying products with images, names, prices, and stock status
- Implement backend data model for products (id, name, description, price, stock quantity, category, image reference)
- Build product detail page with full information, larger image, description, and quantity selector
- Add shopping cart functionality for adding products, adjusting quantities, and viewing totals
- Create backend order management system storing customer information, order items, totals, and status tracking
- Build checkout flow with cart review, shipping information form, and order placement
- Create order history page showing past orders with details and status
- Implement product search and category filtering
- Add inventory management with stock tracking, low-stock alerts, and out-of-stock indicators
- Render generated product images as static assets in frontend public directory

**User-visible outcome:** Users can browse a product catalog, search and filter products, view product details, add items to a shopping cart, complete checkout with shipping information, place orders, and view their order history. Resellers can track inventory levels with visual low-stock indicators.
